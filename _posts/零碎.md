- 函数内可以直接给全局undefined的对象赋值
```javascript
var a = function(){
    abc = "abc"
    var cde = "cde"
}
a()
console.log(abc) // "abc"
console.log(cde) // "cde is not defined"
```

- for in .. 同if的区别

```javascript
'aa' in window.aa // false
if(window.aa)     // false
window.aa = null
'aa' in window.aa // true
if(window.aa)     // false
```
